Font2DTest

An encompassing font/glyph demo application. Source code is in Font2DTest.java. You can run this program either as an applet or as an application.

To run it as an application, execute the Font2DTest class in a 1.3-compatible Java runtime system. For example: 

% java -jar Font2DTest.jar

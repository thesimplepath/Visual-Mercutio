SOAP Feeder

 

Généralités

 

Afin de faciliter l’envoi des messages pour les entreprises ne pouvant pas utiliser directement le service Web de Monitor, un utilitaire d’envoyer le contenu d’un fichier xml au service web.

 

Le fichier XML doit être au format défini dans la section précédente. (Généré avec le XML Builder)

 

 

 

 

SOAP URL : url du service web

ALIAS : alias de l’univers devant être utilisé pour recevoir les messages

XML File : chemin vers le fichier XML à envoyer
